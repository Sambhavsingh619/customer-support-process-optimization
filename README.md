# 📊 Customer Support Process Optimization

## 📌 Problem Statement

Customer support operations often face challenges such as inconsistent response times and delays in ticket resolution. This project analyzes support ticket data to identify inefficiencies, measure performance using key metrics, and recommend data-driven process improvements.

## 🎯 Objective

- Analyze support ticket data  
- Measure operational efficiency using KPIs  
- Identify process inefficiencies  
- Recommend improvements  

## 📊 Dataset

- 100 simulated customer support tickets  
- Fields include:
  - Ticket ID  
  - Created Time  
  - First Response Time  
  - Resolution Time  
  - Priority  
  - Issue Type  
  - Agent Name  

## 📈 Key Performance Indicators (KPIs)

- First Response Time (FRT) – Time taken for first response  
- Turnaround Time (TAT) – Total time to resolve a ticket  
- Resolution Rate – Percentage of resolved tickets  
- Pending Tickets – Number of unresolved tickets
